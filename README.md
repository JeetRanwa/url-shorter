## Features
### Indirect url: 
use of short url using http://m instead of long url<i>(m=https://youtube.com)</i> </li>
### Direct url: 
Use to set dynamic aliases. <i>u/###</i> (Alias) -> <i>http://github.com/rahulworld/url-shorter</i> (Redirect)</li>
